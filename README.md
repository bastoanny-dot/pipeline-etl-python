# pipeline-etl-python
Pipeline ETL desenvolvido em Python para desafio prático

# Pipeline ETL com Python

Este projeto foi desenvolvido como parte de um desafio prático com foco na construção
de um pipeline ETL (Extrair, Transformar e Carregar), usando Python.

O objetivo é demonstrar o entendimento do fluxo de dados, independentemente do uso
de APIs externas ou ferramentas específicas de IA.

## Contexto do Desafio
Durante o desafio, a API original poderia estar indisponível.
Por isso, os dados foram simulados por meio de um arquivo CSV,
seguindo as orientações propostas.

## Tecnologias Utilizadas
- Python
- Pandas
- Google Colab

## Pipeline ETL

# Extrair (Extract)
Leitura dos dados a partir de um arquivo CSV com informações dos clientes.

#Transformar (Transform)
Geração de mensagens personalizadas para cada cliente,
simulando o uso de IA Generativa com Python.

#Carregar(Load)
Salvamento dos dados transformados em um novo arquivo CSV.

## Estrutura do Projeto
- etl_mensagens_clientes.ipynb
- clientes.csv
- clientes_com_mensagens.csv
- README.md

## Conclusão
Este projeto demonstra, de forma prática, a construção de um pipeline ETL completo
utilizando Python, reforçando conceitos essenciais de Engenharia de Dados.
